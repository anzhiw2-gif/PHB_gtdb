# 验证集统计：基因组数量与蛋白序列数量

**日期**: 2026-07-02 | **数据级别**: HMMer + DIAMOND 高置信度验证集 (N=6,532)

---

## 1. 基因组 vs 序列：为什么是两个数字？

```
1 个细菌基因组 ──可能含有──▶ 0 个 phaZ 基因  (不降解 PHB)
                            1 个 phaZ 基因  → 1 条 PhaZ 蛋白
                            3 个 phaZ 基因  → 3 条 PhaZ 蛋白 (旁系同源)
                           12 个 phaZ 基因  → 12 条 PhaZ 蛋白 (基因复制)
```

- **基因组数** = 含有 phaZ 的细菌种类计数（去重）
- **蛋白序列数** = 所有 phaZ 基因翻译出的 PhaZ 蛋白总数（含旁系同源）

---

## 2. 验证集核心统计

| 指标 | 数值 | 说明 |
|---|---|---|
| **验证 PhaZ 蛋白序列** | **6,532** | 通过 HMMer profile + DIAMOND 双重验证 |
| **含 phaZ 的基因组** | **5,531** | 这些蛋白来自多少个细菌 |
| **覆盖门** | **13** | Pseudomonadota 占 96.8% |
| **覆盖属** | **1,135** | Paraburkholderia (319) 最多 |
| **平均拷贝数/基因组** | **1.18** | 多数基因组只有 1 个 phaZ |

## 3. 拷贝数分布

| 拷贝数 | 基因组数 | 占比 |
|---|---|---|
| 1 | 4,682 | 84.7% |
| 2 | 724 | 13.1% |
| 3 | 104 | 1.9% |
| 4 | 17 | 0.3% |
| 5-6 | 4 | 0.07% |

**84.7% 的 phaZ+ 基因组仅含 1 个 phaZ 拷贝**，多拷贝基因组仅占 15.3%。

> ⚠️ 原始 DIAMOND 搜索结果中 67.7% 的基因组表现为多拷贝（最高 12 个），但经 HMMer 验证后大量弱命中被剔除，最终验证集以单拷贝为主。这提示**初始 DIAMOND 搜索中的"多拷贝"很大部分是假阳性同源片段**。

## 8. 与 gRodon2 分析的衔接

gRodon2 使用了 8,788 个基因组（4,394 phaZ+ 配对 4,394 同属 phaZ-），而非仅 5,531 个验证基因组。这是因为：(1) phaZ+ 包含 c95 级别基因组以扩大配对池；(2) phaZ- 对照来自同属但不含 phaZ 的基因组。论文方法中应注明 gRodon2 的基因组集与验证集的差异。
