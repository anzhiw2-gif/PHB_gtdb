# PHB_gtdb — 分析管线

## 核心设计原则

PhaZ (PHB depolymerase, EC 3.1.1.75) **不是单拷贝同源基因**，而是一个 identity 跨度 30%–99% 的 α/β 水解酶超家族。因此：

> - **c95 去真冗余** — 仅移除近完全相同 (≥95% identity) 的序列，保留全部功能多样性
> - **按结构域类型分型** — 基于 Pfam 结构域组成 + 参考序列匹配分为 5 个亚型
> - **同型子集各自建树** — 每个亚型独立 MSA + 系统发育，不强行混合不同类别的序列
> - **c70 仅用于概览可视化** — 粗粒度聚类热图，不写入结论作为 phylogenetic evidence

## 结构域特征详解

### 胞外型 (extracellular / extracellular_lemoignei)

```
N ─[信号肽]──[催化域 Type I/II]──[Linker]──[底物结合域 SBD]── C
              │                              │
              ├─ G-X-S-X-G (lipase box)      └─ 结合 PHB 颗粒表面
              ├─ 催化三联体: Ser-Asp-His
              └─ Pfam: Esterase_phb (α/β hydrolase fold)
```

- **Type I 催化域**: 标准 α/β 水解酶折叠, 环状排列变体
- **Type II 催化域**: 与 Type I 同源但结构略有差异
- **SBD**: C 端底物结合域, 负责吸附 PHB 颗粒, 含 cadherin-like 重复
- **分泌**: N 端信号肽引导 Sec/Tat 分泌到胞外

### 胞内型 (intracellular / ralstonia)

```
N ─[催化域 (无 lipase box)]──[C 端延伸]── C
     │
     ├─ 无 G-X-S-X-G 基序
     ├─ 催化残基未知 (非经典 Ser)
     ├─ Pfam: PHB_depoly_PhaZ (TIGR01849)
     └─ 仅降解无定形 PHB 颗粒 (非结晶)
```

- **无信号肽**: 定位在胞内
- **无 lipase box**: 催化机制与胞外型不同
- **底物特异性**: 仅作用于无定形 (amorphous) PHB

### 新型 (bacillus_type)

- 来自革兰氏阳性菌
- **含有 lipase box-like 序列** (G-W-S-M-G), 但与胞外型不同源
- S102 为关键催化残基 (突变后完全失活)
- 代表: Bacillus thuringiensis PhaZ (PDB: 8YNW)

## CD-HIT 实验数据

```
实验条件: CD-HIT v4.8.1, -n 5 -M 16000 -T 30

阈值   序列数   保留率   评价
─────  ─────   ──────   ──────────────────────────────
c95    8,731   99.6%    ✓ 仅去真冗余, 保留全部功能多样性
c90    8,627   98.4%    几乎无效果, 序列多样性极高
c70    6,483   73.9%    删除 26%, 已开始丢失亚家族
c60    4,400   50.2%    删除 50%, 不同亚家族被强行合并
c50    ~2,500  ~28%    删除 72%, 功能信号严重失真

结论: c95 选定。c70 以上阈值仅用于概览热图, 不作为系统发育证据。
```
