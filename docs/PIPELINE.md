## 关键设计决策

### 为什么不能 c50/c60 去冗余
PhaZ 不是单拷贝同源基因，而是 identity 从 30% 跨到 99% 的酯酶功能超家族。
- **c95** → 仅去真冗余 (近完全相同的序列)
- **c70** → 最多画概览热图，绝不作为系统发育证据
- **正确做法**: c95 → 分型 → 同型子集各自建树
