# 五张汇报图标题与图注

以下图题和图注可直接用于组会汇报、论文初稿或项目总结。所有图片位于 `figures/nature/`，每张图均提供 PDF、PNG 和 SVG 三种格式。

## Figure 1

**标题：GTDB R232 中 PhaZ 的系统鉴定流程与高置信度筛选漏斗**

**图注：**  
Figure 1 展示本项目从 GTDB R232 代表基因组中鉴定 PHB/PHA depolymerase PhaZ 的完整分析流程和数据筛选过程。左侧流程图概括了从基因组输入、Pyrodigal 基因预测、DIAMOND 同源搜索、CD-HIT c95 去冗余、长度过滤、HMMER 与高置信度 DIAMOND 联合验证，到亚型划分、系统发育分析和可视化的主要步骤。右侧漏斗图展示候选序列如何逐步收敛为最终高置信度集合：细菌初始搜索得到 7,068 个命中基因组和 16,486 个候选命中，最终得到 6,532 条高置信度 PhaZ 蛋白序列。该图强调本项目不是单纯依赖一次同源搜索，而是通过多级过滤和验证建立可靠的 PhaZ 数据集。

## Figure 2

**标题：PhaZ 在细菌门水平的分布及亚型结构**

**图注：**  
Figure 2 总结最终验证 PhaZ 序列在不同细菌门中的分布，并展示主要门类中的 PhaZ 亚型组成。结果显示，PhaZ 在分类学上高度不均匀，主要富集于 Pseudomonadota，同时也分布于多个其他细菌门。热图展示不同门中各亚型的相对贡献，说明 PhaZ 的分布不仅存在数量差异，也存在明显的亚型偏好。该图支持 PhaZ 具有分类学集中性和亚型分化两个核心结论。

## Figure 3

**标题：五类 PhaZ 亚型组成与 lipase box 保守基序验证**

**图注：**  
Figure 3 展示最终 PhaZ 验证集中五类亚型的组成比例，并比较各亚型中 lipase box 保守基序的检出情况。PhaZ 亚型包括一般胞外型、*Paucimonas lemoignei* 相关胞外型、*Cupriavidus* 相关胞内型、*Ralstonia* 相关胞内变体和 *Bacillus* 相关新型。lipase box 是许多 PhaZ/水解酶催化活性相关的保守序列特征，胞外型和 Bacillus-like 亚型中检出比例较高，而胞内型中比例较低，提示不同亚型可能存在不同的结构特征和催化基序保守性。该图为亚型划分提供了序列基序层面的支持。

## Figure 4

**标题：PhaZ 富集属分布与亚型系统发育概览**

**图注：**  
Figure 4 从属水平和系统发育角度概括 PhaZ 的分布模式。Top 属分布图显示 PhaZ 并非均匀分散于所有细菌，而是在若干属中明显富集。系统发育概览图展示不同亚型 PhaZ 的分支结构，说明主要亚型之间存在稳定的谱系分化。该图连接了分类学分布和序列演化证据，支持”PhaZ 在特定类群中扩张，并形成多个可区分亚型”的解释。

## Figure 5

**标题：gRodon2 同属内 phaZ+ 与 phaZ- 预测最大生长速率比较**

**图注：**  
Figure 5 展示基于 codon usage bias 的预测最大生长速率在同属内 phaZ+ 与 phaZ- 菌株之间的比较结果。分析覆盖 899 个属、8,692 个平衡配对基因组。左侧面板展示各属内 phaZ+ 与 phaZ- 均值差异（δ = mean(phaZ+) − mean(phaZ−)），右侧面板展示 genome-level 生长速率分布。统计检验显示属均值 δ = −0.0027 growth/h（95% bootstrap CI: −0.0155 到 +0.0093），Wilcoxon 符号秩检验 p = 0.459，分层置换检验 p = 0.670，效应量 r = 0.025。结果表明，在当前数据范围内，同属内携带 phaZ 与不携带 phaZ 的菌株在预测最大生长速率上无显著差异，效应量可忽略。该分析使用 Pfam 核糖体蛋白 HMM 作为 highly expressed genes，gRodon2 预测每个基因组的最大生长速率。注意 gRodon2 输出为基于 codon usage bias 的预测值，不等同于实际培养条件下的测量生长速率。
